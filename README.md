# Visme Unified Growth Dashboard

Self-contained HTML dashboard combining Metabase, GA4, Google Ads, and Microsoft Ads data for weekly executive review.

- **Tab 1 — Growth:** MTD actuals vs projection for PPC, Organic, and New MRR, plus a free-trial pipeline section and weekly/prior-month tables.
- **Tab 2 — Conversions & MRR:** 8w / 6mo / 12mo conversion counts and new MRR by plan.

Rebuilt weekly by `build_growth_dashboard.py` and pushed to this repo. The live dashboard is `index.html`.
